# API 文档

## 抖音

以下POST请求参数均使用raw json格式，例如添加账号接口参数为：`{"id": "test", "cookie": ""}`

### 添加账号

- **功能说明**

用于添加抖音账号，cookie最好复制视频详情页中`https://www.douyin.com/aweme/v1/web/aweme/detail/`接口请求时带的cookie，成功率较高。

- **URL**

  `/douyin/add_account`

- **Method**

  `POST`

- **Data Params**

| 参数 | 必选 | 类型 | 说明 |
|:---:|:---:|:---:|:---:|
| id | true | string | 账户名(用于管理用户cookie) |
| cookie | true | string | 抖音cookie |

- **Response**

| 参数 | 必选 | 类型 | 说明 |
|:---:|:---:|:---:|:---:|
| code | true | int | 0: 成功 1: 参数错误 2: 服务器错误 |
| data | true | struct | 数据 |
| msg | true | string | 请求说明(成功、参数错误、服务器错误) |

### 获取账号列表

- **URL**

  `/douyin/account_list`

- **Method**

  `GET`

- **URL Params**

  None

- **Response**

| 参数 | 必选 | 类型 | 说明 |
|:---:|:---:|:---:|:---:|
| code | true | int | 0: 成功 1: 参数错误 2: 服务器错误 |
| data | true | list | [ [账户信息](#账户信息) ] |
| msg | true | string | 请求说明(成功、参数错误、服务器错误) |

#### 账户信息

| 参数 | 必选 | 类型 | 说明 |
|:---:|:---:|:---:|:---:|
| id | true | string | 账户名(用于管理用户cookie) |
| cookie | true | string | 抖音cookie |
| ct | true | int | 创建时间戳 |
| ut | true | int | 更新时间戳 |
| expired | true | int | 0: 有效 1: 过期 (请求失败时自动设为过期) |

数据库相关:
使用 SQLite 数据库存储账号信息
创建 accounts 表存储账号、cookie 等信息
接口实现:
添加账号 (POST /add_account)
过期账号 (POST /expire_account)
获取账号列表 (GET /account_list)
获取视频详情 (GET /detail)
获取视频评论 (GET /comments)
获取评论回复 (GET /replys)
关键词搜索视频 (GET /search)
用户信息及作品获取 (GET /user)
错误处理:
统一的错误码：
0: 成功
1: 参数错误
2: 服务器错误
3: 业务错误（如获取视频信息失败）
